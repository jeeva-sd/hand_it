*Module 1 — Auth*
Magic link OTP login. Session via JWT. Done.

---

*Module 2 — Workspace*
Create workspace. Store in session. Workspace switcher.

---

*Module 3 — Projects*
Create project. List projects. Project detail page with 4 tabs — Files, Shares, Requests, Logs.

---

*Module 4 — Files*
Upload files to R2. List files inside project. File preview. Delete.

---

*Module 5 — File Versioning*
Upload new version to existing file. Version history. Always show latest by default.

---

*Module 6 — Shares*
Create share from project files. Generate unique public token. Share settings — password, expiry, download toggle.

---

*Module 7 — Public Share Page*
/s/[token] — no login required. Client sees files, previews, downloads. This is your most important page.

---

*Module 8 — Client Actions*
On share page — comment, approve, request changes. No signup needed. Just name + optional email.

---

*Module 9 — Requests*
Freelancer sends file request link. Client opens, uploads files back. Status updates to fulfilled.

---

*Module 10 — Logs*
Every action recorded — opened, downloaded, commented, approved, uploaded. Timeline view inside project.

---

*Module 11 — Notifications*
Email via Resend on every client action. Freelancer gets notified instantly.

---

*Module 12 — Members*
Invite team members to workspace. Owner and member roles only.

---

*Module 13 — Billing*
Paddle integration. Workspace level subscription. Free and Pro plans. Upgrade wall on limits.
